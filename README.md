# 📸 MOA
**"흩어져 있는 우리만의 네컷 추억을 한곳에, 소셜 아카이빙 플랫폼 MOA"**
<br>

![모아 발표자료_page-0001](https://github.com/user-attachments/assets/162d247e-0f08-4016-b08f-214dd5d14fb1)

## 📍 서비스 개요
MOA는 친구들과 함께 찍은 네컷 사진과 소중한 일상을 기록하고 공유하는 소셜 아카이빙 서비스입니다. 단순한 사진 저장을 넘어, 친구 시스템과 커뮤니티 기능을 통해 추억을 매개로 소통할 수 있는 우리만의 디지털 앨범 공간을 제공합니다.
